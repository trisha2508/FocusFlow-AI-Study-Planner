# 🧠 FocusFlow – AI Study Planner

An intelligent desktop study planner that helps students organize their study schedule, track progress, and stay focused using built-in productivity tools.

---

## 🚀 Features

- 📅 **Smart Study Planning**
  - Generates daily study plans based on deadlines and remaining workload
  - Prioritizes subjects with closer deadlines

- 🔁 **Dynamic Adjustment**
  - Updates remaining hours based on actual study progress
  - Handles missed study time effectively

- 📊 **Analytics Dashboard**
  - Tracks total remaining work
  - Shows completed subjects and deadlines

- ⏱ **Focus Tools**
  - Pomodoro Timer (25 min work / 5 min break)
  - Deep Focus Timer for long sessions

- 💾 **Save & Resume**
  - Automatically saves progress using JSON
  - Resume your plan anytime

- 🖥 **GUI-Based Application**
  - Built using Tkinter
  - Multi-page interface (Setup, Plan, Analytics, Timers)

---

## 🛠 Tech Stack

- **Language:** Python  
- **GUI:** Tkinter  
- **Data Storage:** JSON  
- **Concepts Used:**
  - Scheduling logic (deadline-based prioritization)
  - State management
  - File handling
  - Event-driven programming

---

## 🎯 How It Works

1. Add subjects with required study hours and deadlines  
2. Set daily available study time  
3. The app generates a daily study plan  
4. Log your actual study progress  
5. End the day → plan updates automatically  
6. Track your progress in the Analytics page
